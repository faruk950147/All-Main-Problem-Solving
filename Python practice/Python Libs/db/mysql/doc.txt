Step 2: Select Database
Like USE database_name; in MySQL CLI, you can select a database in Python by using the database parameter in the connect() function.

python
conn = mysql.connector.connect(
    host="localhost",
    user="root",
    password="",
    database="python_releases"  # This is like USE your_db_name; in MySQL CLI.
)

Step 3: Cursor (cursor) creation
python
cursor = conn.cursor()
This is used to execute SQL commands.

Step 4: SQL Command Execution
python
cursor.execute("SELECT * FROM python_releases")
result = cursor.fetchall()
for row in result:
    print(row)

# Close connection
cursor.close()
conn.close()

Example of full usage process
python
import mysql.connector

# Server connection + Database usage
conn = mysql.connector.connect(
    host="localhost",
    user="root",
    password="",
    database="python_releases"  
)

# Cursor is a database connection object
cursor = conn.cursor()

# SQL command execution and fetch result
cursor.execute("SELECT * FROM python_releases")
for row in cursor.fetchall():
    print(row)

# Close connection
cursor.close()
conn.close()
